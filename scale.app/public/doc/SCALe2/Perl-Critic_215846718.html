SCALe2 : Perl Critic

Created by Wiki Administrator on Dec 04, 2017

Perl::Critic is an open-source static analysis tool. Perl::Critic's different categories of diagnostics are documented  on Perl::Critic's website .

Here is the command to run Perl::Critic on all files ending in .pl in the current directory:

perlcritic --severity 1 --force --verbose "^^ %p ^^ %f ^^ %l ^^ %m / %e / %s ^^\n" *.pl > perlcritic.txt

If your Perl files exist in multiple directories, you'll want to specify them all on the command line. Providing a directory name should cause Perl::Critic to analyze all Perl files in that directory and its subdirectories.

Formatting Output For SCALe

The resulting file needs to be sanitized to be correctly handled by SCALe. These commands sanitize the file:

perl -pi -e 's/::/_/g;' perlcritic.txt
perl -p -e 's/\|/BAR/g;' perlcritic.txt > perlcritic_results.txt
