SCALe2 : Lizard

Created by Lori Flynn, last modified on Jan 24, 2018

The Lizard code metrics tool is open-source and cost-free, with the github project location at https://github.com/terryyin/lizard

We provide a version of Lizard with the SCALe distribution in the scale.app/scripts/lizard directory)

Use our scale.app/scripts/lizard_metrics.py script to run the provided version of Lizard on the codebase you are analyzing, to output a .csv file with all the needed fields for SCALe's output uploader.

usage: lizard_metrics.py [-h] [-p PATHNAME]

Gathers metrics via Lizard

optional arguments:

  -h, --help            show this help message and exit

  -p PATHNAME, --pathName PATHNAME           Path in which to begin gathering metrics

Lizard can be used on C, C++14, C#, Java, Python, JavaScript, and many more languages (follow hyperlink for details).
