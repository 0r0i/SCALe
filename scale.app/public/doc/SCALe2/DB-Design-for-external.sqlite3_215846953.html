SCALe2 : DB Design for external.sqlite3

Created by Wiki Administrator, last modified by Lori Flynn on Jan 09, 2018

The external.sqlite3 database holds data for all SCALe projects. It is in $SCALE_HOME/scale.app/db/external.sqlite3.

It is used by an intermediate step in importing or exporting a SCALe database. That is why its internal schema matches those of the imported/exported databases. The file is unused at all times except import/export.
