SCALe2 : mongodb 2.6.12 Installation on Ubuntu

Created by Wiki Administrator on Dec 04, 2017

You can install mongo 2.6.12 (the same version running on akehurst) from apt using the following commands. You will need to make sure that apt is setup to work with the SEI proxy.

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 7F0CEB10  
echo 'deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen' | sudo tee /etc/apt/sources.list.d/mongodb.list
sudo apt-get update
sudo apt-get install mongodb-org=2.6.12 mongodb-org-server=2.6.12 mongodb-org-shell=2.6.12 mongodb-org-mongos=2.6.12 mongodb-org-tools=2.6.12

The server should start automatically. You can connect to the local instance with the "mongo" application.

wsnavely@wsnavely:~$ mongo
MongoDB shell version: 2.6.12
connecting to: test
Welcome to the MongoDB shell.
For interactive help, type "help".
For more comprehensive documentation, see
    http://docs.mongodb.org/
Questions? Try the support group
    http://groups.google.com/group/mongodb-user
> show dbs
admin  (empty)
local  0.078GB
> exit
bye

The configuration file for the server is /etc/mongod.conf, by default.
