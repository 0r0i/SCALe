SCALe2 : CCSM

Created by Lori Flynn, last modified on Jan 19, 2018

C Code Source Metrics (CCSM) is an open-source, cost-free tool with a public project repository here: https://github.com/bright-tools/ccsm

CCSM is built as an LLVM tool. clang parses the C/C++ into an abstract syntax tree, then CCSM processes the AST to gather metrics.

CCSM can be used on C/C++ code.
